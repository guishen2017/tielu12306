# tielu12306
12306自动登录，购票<br>


# 运行环境
* 开发语言：python 3.5<br>
* 系统： Windows/Linux
## 依赖
pip install -r requirements.txt


# 使用说明
1.配置conf.py<br>
<br>
  user = "你的账号"  
  password = "你的密码"  
  from_station = "出发地"  
  to_station = '目的地'  
  trian_date = "你要出行的日期"#2018-06-05  

2.购票成功后，登录12306网站，完成支付
